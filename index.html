  • Clientes externos nunca possuem permissão de excluir dados
  • Permissões são herdadas do grupo — não podem ser individualizadas por usuário

  • A interface adapta os menus visíveis conforme o perfil do usuário logado
  • Todas as ações são registradas no Log de Acessos

📦 Exportar Versão do Sistema

Gera um arquivo de texto com o código completo do sistema, carimbo de data/hora e código de versão — para ser salvo e restaurado futuramente pelo Antigravity.

Clique em "Gerar" para ver o código de versão...

🔄 Como Restaurar uma Versão

Para restaurar o sistema a uma versão anterior via Antigravity:

  1. Abra uma conversa no Antigravity
  2. Cole o código de versão exportado (arquivo .txt)
  3. Diga: "Restaurar esta versão do sistema"
  4. O Antigravity irá aplicar o código ao index.html e fazer deploy automaticamente

💡 Guarde os arquivos exportados no OneDrive para acesso rápido
